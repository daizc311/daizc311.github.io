常用的储存库

要注意的坑

  • 储存库要启用了才有效
  • yum repo 只能显示已启用的储存库 显示全部需要yum repolist all
  • 如果使用yum-config-manager配置仓库一定要记得保存
  • 配置完后记得要yum makecache建立缓存，如果不放心可以先yum clean all清除所有缓存
